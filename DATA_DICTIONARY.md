# 📖 S60/M60 拌合料盤資料字典 (Data Dictionary)

> **對應檔案**: `S60_M60_dataset_v8_shared.csv`  
> **單位說明**: 除特別標註外，重量單位皆為 kg，強度為 kgf/cm²，電流為 Amps。

---

## 1. 識別與目標 (Identification & Target)

| 欄位名稱 | 說明 | 備註 |
|:---|:---|:---|
| **match_key** | 拌合批次唯一識別碼 | 格式為時間戳 yyyyMMddHHmmss |
| **母體** | 混凝土配方類型 | S60 (一般管, 3分石), M60 (推進管, 6分石) |
| **日期** | 生產日期 | yyyy-MM-dd |
| **平均強度** | 28天抗壓強度平均值 | 目標變數 (Target). 單位: kgf/cm² |

---

## 2. 拌合電流特徵 (Mixer Current Features) ⚡

**這是本研究最關鍵的特徵群，反映了混凝土在拌合機內的真實流變狀態。**

| 欄位名稱 | 物理意義 | 對強度的影響 |
|:---|:---|:---|
| **平均電流** | 整個拌合過程的馬達負載平均 | 整體能耗指標 |
| **最大電流** | 拌合過程中的峰值負載 | 反映初期骨材堆積阻力 |
| **電流標準差** | 電流波動程度 | 反映攪拌均勻度或料況不穩程度 |
| **最終電流** ⭐ | **卸料前的最終負載電流** | **M60 關鍵指標**。電流高 → 料乾/黏滯度高 → 水灰比低 → 強度高 |

## 3. 配方特徵 (Recipe Features)

| 欄位名稱 | 原料名稱 | 說明 |
|:---|:---|:---|
| **耗用骨材1** | 3分石 | 粗骨材 (S60 主要骨材) |
| **耗用骨材2** | 6分石 | 粗骨材 (M60 專用，粒徑較大) |
| **耗用骨材3** | 砂 | 細骨材 |
| **耗用水** | 水 | 拌合用水量 |
| **耗用泥料1** | 水泥 (Cement) | 主要膠結材 |
| **耗用泥料2** | 矽灰 (Silica Fume) | **M60 關鍵強塑劑**，極細填充材 |
| **耗用泥料4** | 爐石粉 (Slag) | 輔助膠結材 |

---

## 4. 衍生特徵 (Derived Features)

| 欄位名稱 | 計算公式 / 定義 | 物理意義 |
|:---|:---|:---|
| **水灰比** | 水 / 水泥 | 經典強度定律，越低強度越高 |
| **水膠比** | 水 / (水泥+矽灰+爐石) | 現代混凝土強度指標 |
| **膠結料總量** | 水泥 + 矽灰 + 爐石 | 總膠結材用量 |
| **漿體比例** | 漿體體積 / 總體積 | 漿體多則孔隙填充好，強度高 |
| **矽灰取代率** | 矽灰 / 膠結料總量 | 矽灰佔比越高，M60 強度通常越高 |
